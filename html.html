Le HTML

Balise mise en évidence :

  • Texte marqué = <mark>
  • Texte mis en évidence moyenne = <em>
  • Texte mis en évidence forte = <strong>

Les listes :

  • Liste non ordonnée = <ul>
  1. Liste ordonnée = <ol>
  • Chaque ligne = <li>

Les liens :

  • Balise de lien = <a href="what.html">
  • Attribut "title="xxx"" pour la bulle d'aide
  • Attribut "target=""" pour ouvrir le lien de différente façon :
    • target="_blank" : nouvelle fenetre ou onglet
    • target="_self" : meme frame que le lien cliqué
    • target="_parent" : dans le cadre parent (du niveau immédiatement supérieur)
    • target="_top" : Ouvre les liens dans le niveau le plus haut de la fenêtre courante (en remplaçant tous les encadrements)
    • target="nomdelaframe" : ouvre le lien dans la frame nomée
  • Créer une ancre avec "id" dans la balise d'origine
  • what.html peu être un lien vers un autre site = www.google.fr
  • what.html peu être un lien vers une page de notre site = page2.html
  • what.html peu être un lien vers un sous dossier = pages/page3.html
  • what.html peu être un lien vers un dossier parent = ../page4.html
  • what.html peu être un lien vers une ancre = #nomdelancre

Les images :

  • Balise d'image = <img src="image.jpg" alt="nom de l'image" />
  • Balise title pour bulle d'information
  • On peu mixer les éléments :
    • <a href="image2.jpg"><img src="image1.jpg" alt="nom de l'image" title="titre de l'image" />
    • Ceci donne une image cliquable vers autre chose (ici une autre image)

Les balises structurantes:

Visuel des balises structurantes

  • <header> l'en-tete (logo, bannière, slogan...)
  • <footer> le pied de page (Liens de contacts, nom de l'auteur, mentions légales,...)
  • <nav> le menu
  • <section> sert à regrouper des contenus en fonction de leur thématique. Elle englobe généralement une portion du contenu au centre de la page.
  • <aside> conçue pour contenir des informations complémentaires au document que l'on visualise. Ces informations sont généralement placées sur le côté (bien que ce ne soit pas une obligation)
  • <article> sert à englober une portion généralement autonome de la page. C'est une partie de la page qui pourrait ainsi être reprise sur un autre site. C'est le cas par exemple des actualités
