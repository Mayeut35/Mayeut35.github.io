Le HTML

Balise mise en évidence :

  • Texte marqué = <mark>
  • Texte mis en évidence moyenne = <em>
  • Texte mis en évidence forte = <strong>

Les listes :

  • Liste non ordonnée = <ul>
  1. Liste ordonnée = <ol>
  • Chaque ligne = <li>

Les liens :

  • Balise de lien = <a href="what.html">
  • Attribut "title="xxx"" pour la bulle d'aide
  • Attribut "target="blank"" pour ouvrir le lien dans une autre page

  • Créer une ancre avec "id" dans la balise d'origine
  • what.html peu être un lien vers un autre site = www.google.fr
  • what.html peu être un lien vers une page de notre site = page2.html
  • what.html peu être un lien vers un sous dossier = pages/page3.html
  • what.html peu être un lien vers un dossier parent = ../page4.html
  • what.html peu être un lien vers une ancre = #nomdelancre

Les images :

  • Balise d'image = <img src="image.jpg" alt="nom de l'image" />
  • Balise title pour bulle d'information
  • On peu mixer les éléments :
    • <a href="image2.jpg"><img src="image1.jpg" alt="nom de l'image" title="titre de l'image" />
    • Ceci donne une image cliquable vers autre chose (ici une autre image)
