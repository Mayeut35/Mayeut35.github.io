Le CSS

Mise en place du CSS dans le ficher HTML :

Il faut inclure la ligne de code suivante dans la partie <head> de la page html

  • <link rel="stylesheet" href="index.css">

Généralités :

Notes :

Il est possible de mettre des notes non prises en comptes par le navigateur
Il faut pour cela mettre le texte entre /* et */

Fonctionnement général :

balise1
{
propriete1: valeur1;
propriete2: valeur2;
propriete3: valeur3;
}

balise2
{
propriete1: valeur1;
propriete2: valeur2;
propriete3: valeur3;
}

L'attribut class :

Il est possible d'ajouté l'attribut class à une balise afin de pouvoir agir uniquement sur les balises de cet attribut avec le CSS.
  • <h3 class="titrerouge"></h3>
  • .titrerouge pour le selectionner sur CSS

Les balises universelles :

Il existe 2 balises qui ne servent qu'à isolé une partie d'un texte, ou tout un bloque, sans passé par les balises génériques.
  • <span>, balise inline qui permet d'isoler une partie d'un paragraphe par exemple
  • <div>, balise block, qui entoure un bloc de texte entier
  • On notera <span class="attribut">partie à isoler<span>

Les sélécteurs avancés :

  • h1, em signifie que les 2 balises sont prises en comptes
  • .attribut pour agir uniquement sur l'attribut
  • * agit sur toutes les balises
  • h1 em agit sur les em inclut dans des h1
  • h1 + p agit sur le 1er p situé après un h1
  • a[title] agit sur les liens a qui ont un attribut title
  • a[title="clic"] agit sur les liens a qui ont un attribut title dont la valeur exact est clic
  • a[title*="clic"] agit sur les liens a qui ont un attribut title dont la valeur contient clic

Formatage du texte :

La police :

Les polices courantes :

Arial, Arial Black, Comic Sans MS, Courier New, Georgia, Impact, Times New Roman, Trebuchet MS, Verdana.

Intégration :

Il faut utiliser font-family: Police;

Utiliser une police autre

  • Force le navigateur de l'utilisateur à télécharger la police
  • Veiller à utiliser des polices libre de droit sur www.fontsquirrel.com par exemple.
  • Utiliser plusieurs formats d'une même police pour assurer la compatibilté des navigateurs web, smartphone, etc...
    • .ttf IE9 et autres nav
    • .eot IE uniquement
    • .otf Pas sur IE
    • .svg Seul format pour Iphone / Ipad
    • .woff Nouveau format pour IE9 et autres nav
  • Commande particulière pour intégré une police, qui doit se trouver dans les dossier du site
@font-face {
....font-family: 'MaSuperPolice';
....src: url('MaSuperPolice.eot') format('eot'),
...........url('MaSuperPolice.woff') format('woff'),
...........url('MaSuperPolice.ttf') format('ttf'),
...........url('MaSuperPolice.svg') format('svg');
  • On peu ensuite appelé la police comme pour les autres avec font-family: 'MaSuperPolice';
  • Pour éviter tout problème, il est préférable d'ajouter des polices de bases font-family: 'MaSuperPolice', Arial, "Comic Sans MS";

La taille

  • On utilisera la commande em avec une valeur devant pour modifier la taille du texte
  • On notera : font-size: 1em; pour une taille normale.

Italic, Gras, Souligné...

  • font-style: normal; gere l'italic et l'oblique avec italic et oblique
  • font-weight: normal; gere le gras avec bold
  • text-decoration: none; gere :
    • underline : souligné
    • line-through : barré
    • overline : ligne au-dessus
    • blink : clignotant (pas sur IE ni Chrome)

L'alignement

  • text-align: left; gere l'alignement du texte
    • left : texte à gauche
    • right : texte à droite
    • center : texte centré
    • justify : rend le texte justifié

Les flottants

  • Il est possible d'aligné le texte autour d'une image avec float: left; ou float: right;
  • Float s'utilise avec la balise de l'image, qui devra donc avoir une class
  • On peu stopper le flottant avant la fin de l'image avec clear: both; appliqué sur le paragraphe qui doit suivre sous l'image

La couleur de la police :

  • On utilse la commande color: maroon;
    • On peu noter le nom de la couleur (maroon, red, green, etc...) mais c'est limité
    • On peu utiliser l'hexadecimale color: #FF5A28;
    • On peu utiliser le RGB color(240,96,204);
    • Le site www.colorpicker.com permet de trouver facilement les références hexadecimales ou rgb d'une couleur voulue

La couleur du fond :

  • On utilise la commande background-color: black;
  • Pour la couleur de la police, le fond et le reste, c'est la commande la plus précise qui prend le dessus

Le fond :

Appliquer une image de fond :

  • On utilise la commande background-image: url(image.png);

Fixer l'image de fond :

  • On utilise background-attachment: fixed; ou scroll

Répéter le fond :

  • On utilise background-repeat: repeat;
    • no-repeat l'image sera unique
    • repeat-x l'image sera répété uniquement sur l'horizontale
    • repeat-y l'image sera répété uniquement sur la vertivale
    • repeat l'image sera répété en mosaïque (par défaut)

Position du fond :

  • On utilise background-position: top; principalement avec background-repeat:
    • top en haut
    • bottom en bas
    • left à gauche
    • right à droite
    • center au centre
  • On peu cumuler les attributs background-position: top right;

Super propriété :

  • On peu cumuler sur une seule ligne plusieurs propriétés au fond
    • background: url("fond.png") fixed no-repeat top right;
  • On peu aussi cumuler les fonds
    • background: url("fond.png") fixed no-repeat top right, url("fond2.png") fixed;

Transparence :

Transparence de tout objet :

  • On utilise opacity: 0.3; avec 1 pour 100%

Transparence du fond :

  • On préfèrera la notation RGBa pour gérer la transparence du fond
    • background-color: rgba(255,0,0,0.5); /* Fond rouge à moitié transparent */

Les bordures :

Bordures standars :

  • Super propriété border: largeur couleur type;
    • Largeur en pixel : 2px
    • Couleur comme pour les polices
    • Type :
      • None pas de bordure
      • Solid simple trait
      • Dotted pointillés
      • Dashed tirets
      • Double bordure double
      • Groove en relief
      • Ridge autre relief
      • Inset 3D enfoncé
      • Outset 3D surélevé
  • Il est possible de configurer chaque côté de la bordure avec border-top, bottom, left, right

Bordures arrondies :

  • On utilise border-radius: 10px;
  • On peu gérer chaque coin de la bordure border-radius: 10px 5px 5px 2px;

Les ombres :

Ombres sur boites :

  • On utilise box-shadow: 2px 2px 2px black (inset); qui gère dans l'ordre :
    • Décalage horizontal
    • Décalage vertical
    • Adoucissement du dégradé
    • Couleur de l'ombre
    • Inset place l'ombre dans le cadre

Ombre sur texte :

  • On utilise text-shadow: 2px 2px 2px black; avec le même ordre que pour les boites.

Apparences dynamiques :

  • Il est possible de modifier l'apparence d'un élément en fonction que l'on passe la sourie dessus ou que l'on clic.
  • On utilise alors des pseudo-formats que l'on ajoute après la balise dans le CSS.
    • Pour le survol on utilise a:hover => www.google.fr
    • Au moment du clic on utilise a:active => www.google.fr
    • A la selection on utilise a:focus => www.google.fr
    • Pour un lien déjà visité, on utilise a:visited => www.google.fr
